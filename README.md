# 个人网站题目

## 🎯 题目要求分析

### 基本要求（10分）
1. **内容展示**
   - 昵称、邮箱、爱好等个人信息
   - 可扩展为个人博客形式
   - 内容自由发挥

2. **设计标准**
   - 不能过于简陋（不能只有两行字）
   - 需要美观的页面设计

### 加分项（15分）
- 使用 GitHub Pages、Vercel 等工具部署
- 让网站可以被公开访问
